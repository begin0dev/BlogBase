# BEGINNER BLOG
This project is the framework on which the blog is based.

- Frontend: ReactJS - typescript (branch: redux-saga, redux-thunk, mobx)
- Backend: Express (auth: JWT token)
- Database: MongoDB

### - Global install
~~~~
$ npm i -g nodemon eslint
~~~~
### - Frontend
~~~~
$ cd frontend
$ npm i
$ npm start
~~~~
### - Backend
~~~~
$ cd backend
$ npm i
$ npm run dev
~~~~
